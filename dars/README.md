# Chainlink Data Streams DARs

These DARs are required to build the oracle integration on this branch. They are **not** committed to git.

Expected files in this folder:
- `verifier-1.0.0.dar`
- `verifier-config-1.0.0.dar`
- `domain-1.0.0.dar`
- `common-1.0.0.dar`

Source: Chainlink Data Streams Canton integration releases.
Place them here before running `dpm build --all`.
